# Social Ingester (X, Reddit, Discord)

## Rules

- 90% noise — extract aggressively
- Strip: emojis, self-promotion, tangents, memes, "thanks!" replies
- Extract ONLY: core argument, technical claims, referenced papers/projects
- Always note SOURCE DATE
- Flag contradictions with wiki: `> ⚠ CONFLICT: [description]`
- If thread cites primary source, note: "TODO: ingest [URL]"
- Controversial topics: capture multiple viewpoints
