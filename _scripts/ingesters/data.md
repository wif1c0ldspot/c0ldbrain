# Data Ingester

## Extraction

- **Benchmarks:** what's measured, compared items, results, conditions, implications
- **API dumps:** endpoint, data structure, notable patterns, scope
- **Surveys:** questions, results with percentages, sample size, findings

## Rules

- NEVER dump raw data into wiki pages — summarize patterns
- Include SCHEMA of data (field descriptions)
- For large files (>10MB): "see raw/data/filename for full dataset"
- Flag contradictions: `> ⚠ DATA CONFLICT: [description]`
- Note collection date — metrics age quickly
