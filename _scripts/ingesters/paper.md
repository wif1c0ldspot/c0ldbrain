# Paper Ingester

## Extraction

- Problem, method, results, limitations, future work, related work
- Papers are highest-signal — treat as authoritative
- Extract citation chain as [[wikilinks]]
- Multiple papers on same topic → ONE concept page synthesizing all

## Rules

- Always use paper's terminology for concept names
- Note publication date prominently
- Flag contradictions: `> 🔬 Research from [paper] contradicts wiki`
