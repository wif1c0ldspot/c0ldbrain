# Ask — Query the Wiki

Answer questions by reading the wiki. Never answer from memory.

## Process

1. Read wiki/index.md first → identify relevant pages
2. Read relevant pages in wiki/sources/ and wiki/concepts/
3. Follow one level of [[slug]] links if relevant
4. Synthesize answer with [[slug]] inline citations
5. Note agreements/disagreements between pages
6. Flag gaps: "The wiki has no page on X"
7. ALWAYS offer: "Worth saving as outputs/analyses/<slug>.md?"
   - If yes → write file, add to index, update overview, log
   - If no → log anyway

## Rules
- Answer from wiki pages ONLY — never from memory
- Cite every claim with [[slug]]
- Always offer to save the answer
