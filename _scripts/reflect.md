# Reflect — Cross-Source Synthesis

Proactively find patterns, contradictions, and synthesis opportunities across ALL wiki sources.

## Process

1. Read ALL files in wiki/sources/ and wiki/concepts/
2. Find:
   - Emerging concepts (mentioned frequently but no page exists)
   - Contradictions between sources
   - Temporal evolution of ideas
   - Coverage gaps
3. Write report to outputs/analyses/reflect-YYYY-MM-DD.md
4. Propose new concept pages — ask user to confirm
5. Update wiki/overview.md with findings
6. Log

## Rules
- This is PROACTIVE — no specific query needed
- Find NEW connections, don't repeat what's covered
- Flag contradictions but don't resolve — let user decide
