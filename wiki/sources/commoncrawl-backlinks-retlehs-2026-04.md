## Summary

Ben Word (@retlehs) discovered you can extract backlinks to any domain for free using Common Crawl's web graph data — normally a $100-999/month feature on Ahrefs/SEMrush. Wrote a bash script to query the open graph dataset. 458K views on the tweet.

## Key Points

- **Common Crawl web graph**: Free, open hyperlink relationship data (300B+ pages, 15 years)
- **Replaces paid tools**: Ahrefs ($99-999/mo), SEMrush ($119-449/mo), Moz ($99-599/mo)
- **Quarterly updates**: Latest cc-main-2026-jan-feb-mar
- **Host and domain level**: Both granularity available
- **Limitations**: No real-time, no filtering, no metrics (raw data only)
- **Use cases**: Competitive analysis, research intelligence, link building, trend analysis
