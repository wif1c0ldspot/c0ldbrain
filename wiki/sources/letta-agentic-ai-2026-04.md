# Letta

## Summary

Letta (github.com/letta-ai/letta) is a platform for building stateful AI agents with advanced memory that can learn and self-improve over time. 21.9k GitHub stars. Evolved from the MemGPT research project.

## Source

- https://github.com/letta-ai/letta

## Key Features

- Stateful agents — agents persist across sessions with long-term memory
- Advanced memory architecture — archival, recall, and working memory
- Self-improvement — agents learn and adapt over time
- MCP integration — connect to external tools and services
- Multi-provider LLM support

## Architecture

- Memory management with hierarchical storage
- Agent state persistence across conversations
- Tool calling and function execution
- Cloud and self-hosted deployment options
