# GenericAgent: Token-Efficient Self-Evolving LLM Agent

## Summary

GenericAgent proposes "Contextual Information Density Maximization" — a token-efficient approach to self-evolving agent memory. Addresses the fundamental context overflow problem for long-horizon agents by maximizing information density in the context window while retaining useful experience.

## Key Details

- **ArXiv ID:** 2604.17091v1
- **Authors:** Jiaqing Liang, Jinyi Han, Weijia Li, et al. (+15)
- **Date:** 2026-04-18
- **Categories:** cs.CL

## Key Contributions

1. **Contextual Information Density Maximization** — maximize useful information per token
2. **Self-evolving memory** — agents learn and compress experience over time
3. **Context overflow solution** — addresses the fundamental scaling problem for long-horizon agents

## Notes

- Directly relevant to the "memory quality > memory quantity" thesis
- Complementary to cavemem's compression approach but focuses on density, not just reduction
- The "information density" framing is novel and actionable
