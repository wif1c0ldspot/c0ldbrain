# Microsoft AutoGen

## Summary

AutoGen (github.com/microsoft/autogen) is a programming framework from Microsoft for building agentic AI systems. It enables multiple AI agents to converse and collaborate to solve complex tasks. Over 56.9k GitHub stars — the most starred agentic AI repository.

## Source

- https://github.com/microsoft/autogen

## Key Features

- Multi-agent conversation framework — agents talk to each other to solve tasks
- Human-in-the-loop support — humans can participate as agents
- Code execution agents — agents can write and run code autonomously
- Customizable agent types — assistant, user proxy, group chat managers
- Supports multiple LLM backends (OpenAI, open-source, local)
- Async and sync APIs

## Architecture

- Agent abstraction layer with conversation patterns
- Group chat orchestrator for multi-agent workflows
- Code executor sandbox for safe code generation/execution
- Extensible tool use and function calling
