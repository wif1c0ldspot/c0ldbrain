# DB-GPT

## Summary

DB-GPT (github.com/eosphoros-ai/DB-GPT) is an open-source agentic AI data assistant for next-generation AI + Data products. Features text-to-SQL, RAG, and multi-agent data workflows. 18.5k GitHub stars.

## Source

- https://github.com/eosphoros-ai/DB-GPT

## Key Features

- Natural language to SQL conversion
- Multi-database support (MySQL, PostgreSQL, SQLite, etc.)
- RAG pipeline for structured and unstructured data
- Multi-agent collaboration for complex data tasks
- Knowledge graph integration
- Visual workflow builder for data pipelines

## Architecture

- Agent orchestration for data tasks
- Text-to-SQL with schema understanding
- Vector database for semantic search over data
- Plugin system for extensibility
