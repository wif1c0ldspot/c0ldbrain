# ActivePieces

## Summary

ActivePieces (github.com/activepieces/activepieces) is an open-source AI agents, MCPs, and workflow automation platform. Provides ~400 MCP servers for AI agents with workflow orchestration capabilities. 21.6k GitHub stars.

## Source

- https://github.com/activepieces/activepieces

## Key Features

- 400+ MCP server integrations for AI agents
- Visual workflow builder for AI automation
- AI agent orchestration with triggers and actions
- Self-hosted and cloud options
- No-code/low-code builder with code extensibility

## Architecture

- Piece-based modular system (each integration is a "piece")
- Event-driven workflow engine
- MCP protocol support for tool discovery and usage
- REST API for programmatic control

## Metrics

- 21,648+ GitHub stars
- Large integration ecosystem

## Notes

- Production-ready automation platform
- MCP integration makes it highly relevant for agentic AI
- Good for connecting AI agents to real-world services
