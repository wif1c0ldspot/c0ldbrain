# HiGMem: Hierarchical and LLM-Guided Memory System

## Summary

HiGMem proposes a hierarchical memory system where the LLM itself makes retrieval decisions rather than relying solely on vector similarity. Accepted to ACL 2026 Findings. Code available at https://github.com/ZeroLoss-Lab/HiGMem.

## Key Details

- **ArXiv ID:** 2604.18349v1
- **Authors:** Shuqi Cao, Jingyi He, Fei Tan
- **Date:** 2026-04-20
- **Venue:** ACL 2026 Findings
- **Code:** https://github.com/ZeroLoss-Lab/HiGMem

## Key Contributions

1. **LLM-guided retrieval** — the model itself decides what memory to retrieve
2. **Hierarchical structure** — multiple levels of memory granularity
3. **Overcomes vector similarity limitations** — vector similarity alone misses relevant context
