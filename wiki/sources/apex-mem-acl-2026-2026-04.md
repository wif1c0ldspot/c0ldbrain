# APEX-MEM: Agentic Semi-Structured Memory with Temporal Reasoning

## Summary

APEX-MEM presents a conversational memory system combining three innovations: (1) property-based memory representation, (2) temporal reasoning, and (3) agentic memory management. Accepted to ACL 2026 Main Conference.

## Key Details

- **ArXiv ID:** 2604.14362v1
- **Authors:** Pratyay Banerjee, Masud Moshtaghi, Shivashankar Subramanian, et al.
- **Date:** 2026-04-15
- **Venue:** ACL 2026 Main Conference
- **Key Innovation:** Semi-structured memory with temporal reasoning — moves beyond flat vector stores to structured, time-aware memory

## Key Contributions

1. **Property-based memory** — structured representation beyond text chunks
2. **Temporal reasoning** — understanding when memories are valid
3. **Agentic management** — the memory system itself acts as an agent deciding what to store/retrieve

## Notes

- Addresses noise and instability from naive retrieval in long conversations
- Part of the broader trend toward structured/hierarchical memory systems
- Complements HiGMem (also ACL 2026) in pushing hierarchical approaches
