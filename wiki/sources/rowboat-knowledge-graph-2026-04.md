# Rowboat — AI Knowledge Graph Coworker

## Key Insights

- AI-powered coworker that automatically builds personal knowledge graphs
- Ingests email, meeting transcripts, and documents to extract structured knowledge
- Outputs Obsidian-compatible vault — direct integration with existing PKM workflows
- 12.4k stars indicate strong community adoption

## Technical Details

### Data Sources
- Email ingestion and parsing
- Meeting transcript processing (Zoom, Teams, Meet)
- Document analysis (PDFs, docs, web pages)

### Knowledge Graph Construction
- Entity extraction and relationship mapping
- Automatic topic clustering and categorization
- Temporal tracking of knowledge evolution
- Bidirectional linking between concepts

### Obsidian Integration
- Generates Obsidian-compatible markdown vault
- Uses wikilinks for cross-referencing
- Frontmatter metadata for filtering and search
- Compatible with existing Obsidian plugins and themes

### Architecture
- LLM-powered extraction pipeline
- Graph database backend for relationship storage
- Incremental updates as new data arrives
- Privacy-first: can run fully locally
