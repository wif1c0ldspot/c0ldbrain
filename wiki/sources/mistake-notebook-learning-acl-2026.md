# Mistake Notebook Learning (MNL)

**ACL 2026 Findings Paper**

A novel, training-free memory framework that enables LLM agents to systematically learn from their mistakes.

## Key Innovation

- **Training-free** — no gradient updates needed
- **Mistake as memory** — errors become durable learning artifacts
- **Systematic** — structured approach to error capture and avoidance

## Camp Classification

**Camp 1.5 Hybrid** — Stores mistakes as a form of long-term memory (Camp 1), but retrieval directly modifies agent behavior at runtime (Camp 2).
