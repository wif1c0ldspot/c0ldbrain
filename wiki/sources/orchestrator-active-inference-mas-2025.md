# Orchestrator: Active Inference for Multi-Agent Systems in Long-Horizon Tasks

**arXiv:** [2509.05651](https://arxiv.org/abs/2509.05651)
**Date:** September 6, 2025
**Authors:** Beckenbauer, Loewe, Zheng, Brintrup

## Key Claims

- Complex non-linear tasks challenge LLM-enhanced multi-agent systems (MAS)
- Partial observability and suboptimal coordination are key problems
- Orchestrator introduces **attention-inspired self-emergent coordination**
- Uses **active inference benchmarks** to optimize system behavior
- Tracks agent-to-agent and agent-to-environment interaction

## Architecture

```
Agents ↔ Orchestrator (Active Inference Monitor)
    → Track dynamics
    → Active inference benchmarks
    → Optimize global task performance
    → Mitigate partial observability
```

## Why This Matters

1. **Self-emergent coordination** — agents organize without central planning
2. Active inference provides principled coordination criterion
3. Applicable to agent delegation patterns (like subagent task decomposition)
4. Attention mechanism maps to neural oscillation synchronization (Day 3 pathway)
