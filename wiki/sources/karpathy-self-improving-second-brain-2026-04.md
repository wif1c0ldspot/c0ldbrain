## Summary

Karpathy describes his personal knowledge base as an "ultimate self-improving second brain" that requires zero manual editing. The system automatically organizes and links information.

## Core Properties

### Zero Manual Editing
- No hand-crafted notes
- No manual linking
- No categorization required
- Agent handles all organization

### Self-Improving Behavior
- Learns from usage patterns
- Refines connections over time
- Consolidates similar content
- Surfaces relevant information proactively

### Human + AI Collaboration
- Human provides raw material (bookmarks, links)
- AI structures and connects
- Human can read/verify
- System grows organically

## Technical Architecture

```
Human Input → Raw Capture → Agent Processing → Structured Output
     ↑                                              ↓
     ←←←←←←←←←←← Feedback Loop ←←←←←←←←←←←←←←←←←←←←
```

## Why It Works

1. **Context compounding** — Each session adds to collective knowledge
2. **Pattern recognition** — AI finds connections humans miss
3. **Living document** — Wiki evolves, not static
4. **Low friction** — Just add content, agent organizes
