# Context Constitution

## Overview

Letta released the Context Constitution (April 2, 2026) — a set of principles governing how AI agents manage context to learn from experience. This represents the most explicit articulation of Camp 2 philosophy: memory IS context management.

## Core Thesis

> "Today's models deeply identify with their own ephemerality. They have no motivation for long-term improvement because they don't believe they persist."

The Constitution defines:
1. **Context forms identity** — what's in the context window shapes the agent's sense of self
2. **Context is scarce** — managing context resources is the core engineering challenge
3. **Learning through token-space** — agents learn by actively reading/writing memory blocks and editing system prompts

## Letta Code App

The Letta Code app (launched April 6, 2026) is the consumer-facing product built on this framework:
- **Git-backed memory** — version-controlled memory
- **Multi-conversation shared memory** — agents share context across conversations
- **Sleep-time compute** — reflection during idle periods
- **Desktop app** — local-first, privacy-preserving

## Camp Classification

**Camp 2 (Context Substrate)** — Letta is the purest context-substrate play. Their entire thesis is that memory IS context management. Agents learn by actively managing their own context (reading/writing memory blocks, editing system prompts).

## Why It Matters

1. **Philosophical framework** — articulates WHY context management matters for agent identity
2. **Git-backed memory** — version control for agent memory is a powerful pattern
3. **Sleep-time compute** — agents reflecting during idle time enables continuous improvement
4. **Open-sourced** — the Constitution is available for other frameworks to adopt
