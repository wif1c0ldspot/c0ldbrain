# OpenViking Context Database

## Overview

OpenViking (from ByteDance/Volcengine) is an open-source context database designed specifically for AI agents. At 22,762 stars (as of 2026-04-22), it's one of the fastest-growing agent infrastructure projects. It unifies memory, resources, and skills management into a single context layer.

## Key Features

- **Unified context management** — memory, resources, skills in one system
- **Context database** — not just a memory store, but a database optimized for agent context
- **Open source** — Python, actively maintained (v0.3.9 released April 18, 2026)

## Camp Classification

**Camp 2 (Context Substrate)** — OpenViking is explicitly a "context database" not a "memory backend." The emphasis on unified management of context (memory + resources + skills) is a substrate-level concern.

## Why It Matters

1. **ByteDance backing** — significant engineering resources behind it
2. **22.7K stars in weeks** — massive adoption velocity
3. **"Context database" framing** — reframes agent memory as a database problem, not a storage problem
4. **Unified approach** — challenges the fragmentation between memory, tools, and skills management
