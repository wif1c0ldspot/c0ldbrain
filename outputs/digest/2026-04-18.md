### Key Finding: Async Subagent Pattern
The LangChain async subagent pattern represents a significant evolution in agent orchestration:

- **Problem Solved**: Deadlocks from inline subagent execution
- **Solution**: Non-blocking task launch with polling/checking mechanisms
- **Memory Implications**: Requires both Camp 1 (persistent storage) and Camp 2 (task isolation)
- **Hermes Status**: Already implements via `delegate_task`
